# Page Optimization Checklist — Single Page Deep Dive

## Title & Meta

- [ ] Title ≤ 60 chars
- [ ] Title includes target keyword (naturally, not forced)
- [ ] Title matches content (no clickbait — check C01)
- [ ] Meta description ≤ 155 chars
- [ ] Meta description includes keyword + CTA

## Structure

- [ ] Single H1 matching page topic
- [ ] H2-H3 hierarchy is logical (no skipped levels)
- [ ] Heading structure forms a complete outline
- [ ] Short paragraphs (2-4 sentences max)
- [ ] Bullet or numbered lists where appropriate

## Content Quality (CORE-EEAT)

- [ ] C01: Title truthfully represents content
- [ ] C02: Direct answer to main query in first 150 words
- [ ] C03: Comprehensive (covers subtopics user expects)
- [ ] C04: Value beyond first-page competitors
- [ ] C05: Appropriate depth for topic
- [ ] C06: Author attribution (name + bio + credentials)
- [ ] C07: Publication date + last updated date
- [ ] C08: Internal links to related content
- [ ] C09: FAQ or Q&A section (if applicable)

## Readability

- [ ] R01: Claims have supporting evidence
- [ ] R02: Citations link to authoritative sources
- [ ] R03: Source hierarchy (prefer primary sources)
- [ ] R04: Every claim has evidence
- [ ] R05: Methodology documented (if research-based)
- [ ] R06: Technical accuracy verified
- [ ] R07: Bias acknowledged
- [ ] R08: Data accurate and consistent
- [ ] R09: Freshness appropriate for topic
- [ ] R10: No data contradictions

## Technical

- [ ] Canonical tag: self-referencing or correct
- [ ] `noindex` NOT present (unless intended)
- [ ] Open Graph tags present
- [ ] Twitter Card tags present
- [ ] Schema markup of correct type
- [ ] Page loads via HTTPS
- [ ] Alt text on all images
- [ ] Mobile responsive
- [ ] Page speed acceptable (subjective if no tool data)

## GEO / AI Visibility

- [ ] C02: Direct answer extractable
- [ ] TL;DR or Key Takeaways box present
- [ ] Structured data present (schema)
- [ ] External citations to authoritative sources
- [ ] Original insight or data point present
