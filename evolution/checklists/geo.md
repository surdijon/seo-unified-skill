# GEO Checklist — AI Search Visibility Audit

## Phase 1: AI Crawler Access

- [ ] `robots.txt` allows GPTBot
- [ ] `robots.txt` allows ClaudeBot
- [ ] `robots.txt` allows PerplexityBot
- [ ] `robots.txt` allows Google-Extended
- [ ] `robots.txt` allows Applebot-Extended
- [ ] `robots.txt` allows CCBot (Common Crawl)
- [ ] `robots.txt` allows FacebookBot (Meta AI)
- [ ] **If ANY AI crawler blocked → fix is HIGH priority**

## Phase 2: Content Extractability

- [ ] C02 pass: Direct answer in first 150 words
- [ ] C03 pass: Comprehensive coverage of topic
- [ ] O02 pass: Key Takeaways / TL;DR box present
- [ ] O03 pass: Data tables in HTML (not images)
- [ ] C09 pass: FAQ schema + visible FAQ section
- [ ] R01 pass: Precise claims (not vague)
- [ ] R02 pass: Outbound citations to authoritative sources
- [ ] C10 pass: Multimedia with alt/figcaption

## Phase 3: Brand Entity / Quotability

- [ ] E01 pass: Original research/data on page
- [ ] R03 pass: Source hierarchy (primary > secondary > tertiary)
- [ ] R04 pass: Claim-to-evidence ratio (every claim backed)
- [ ] O04 pass: Author/team pages with credentials
- [ ] Ept05 pass: Methodological rigor documented
- [ ] Exp10 pass: Limitations and scope stated

## Phase 4: Schema for AI

- [ ] Article schema on all content pages
- [ ] Organization schema (with logo, social profiles)
- [ ] FAQPage schema (if FAQ section exists)
- [ ] Schema matches content type
- [ ] Required fields filled (Google's structured data guidelines)

## Phase 5: llms.txt

- [ ] `/llms.txt` exists
- [ ] Summary is correct and compelling
- [ ] Links cover all important content
- [ ] Format follows llmstxt.org spec
- [ ] Updated monthly

## Phase 6: Scoring

- [ ] Quick citability score calculated (0-100)
- [ ] AI visibility score calculated
- [ ] Top 6 GEO priorities checked
- [ ] GEO report delivered
