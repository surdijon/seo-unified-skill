# SEO·GEO·EEAT-CORE

> Универсальный AI-скилл для аудита и продвижения ваших статических сайтов (Astro/HTML) в топ Google и генеративных ИИ. Ноль настроек. Ноль зависимостей.

## Одной командой

\`\`\`bash
auto https://ваш-сайт.рф
\`\`\`

Скилл запускает 5 фаз диагностики:
1. **Discovery** — загрузка страницы, теги head, технический анализ, скорость
2. **Content** — оценка EEAT по 70 критериям, ключевые слова, бриф
3. **Local** — Google Business Profile, NAP, отзывы
4. **GEO** — видимость в ChatGPT, Perplexity, Claude
5. **Links** — обратные ссылки, конкурентные разрывы

Результат: отчёт с приоритетами, оценкой времени и готовым кодом.

## Все команды

### Запуск
| Команда | Что делает |
|---------|------------|
| \`help\` | Список всех команд |
| \`auto <url>\` | Полный пайплайн (всё в одном) |
| \`audit <url>\` | Аудит 3 уровня + оценка EEAT |
| \`head <url>\` | Проверка 40 тегов head |

### Контент
| Команда | Что делает |
|---------|------------|
| \`keywords <topic>\` | Ключевые слова с интентом поиска |
| \`brief <topic>\` | Редакционный бриф для написания |
| \`content <url>\` | Полная оценка EEAT (70 критериев) |
| \`clusters <topic>\` | План тематического кластера |
| \`fresh <url>\` | Проверка актуальности контента |
| \`recover <url>\` | Восстановление статьи с падающим трафиком |

### Техника
| Команда | Что делает |
|---------|------------|
| \`speed <url>\` | Анализ скорости (LCP, CLS, INP) |
| \`technical <url>\` | Краулинг, индексация, сигналы |
| \`schema <url>\` | Обнаружение и генерация JSON-LD |

### Локальное
| Команда | Что делает |
|---------|------------|
| \`local <business>\` | Аудит Google Business Profile + отзывы |
| \`napcheck <business>\` | Проверка имени/адреса/телефона в 5 каталогах |
| \`links <url>\` | Анализ обратных ссылок |
| \`gap <business>\` | Разрывы с топ-3 конкурентами |
| \`serp <keyword>\` | Стратегия сниппетов + AI Overviews |

### ИИ
| Команда | Что делает |
|---------|------------|
| \`geo <url>\` | Видимость в генеративных ИИ |
| \`citecheck <url>\` | Проверка цитирования сайта ИИ |

## Пример

\`\`\`bash
auto https://santekhnik-dijon.fr
\`\`\`

Скилл:
1. Проверит, могут ли ИИ сканировать сайт (robots.txt)
2. Проанализирует теги head, скорость, схему
3. Оценит контент по 70 критериям (адаптируется под новый сайт)
4. Проверит карточку Google Business Profile
5. Посмотрит, цитирует ли ChatGPT сайт
6. Сравнит с 3 главными конкурентами

→ **Полный отчёт с действиями по приоритету и оценкой времени.**

## Что делает его уникальным

| Другие инструменты | Этот скилл |
|--------------------|------------|
| Произвольные оценки | Каждая оценка обоснована конкретным критерием |
| Игнорируют ИИ | Проверяет ChatGPT, Perplexity, Claude |
| Идеальны для всех сайтов | Адаптируется под зрелость (новый → устоявшийся) |
| Без отслеживания | Непрерывное обучение через evolution/ |
| Сложная настройка | Ноль конфигурации, достаточно одной команды |

## Требования

- ИИ-агент (Claude Code, open-code, Cursor, Cline, Aider, Codex CLI, Gemini CLI)
- Статический сайт (Astro, HTML, Hugo, 11ty и т.д.)
- (Опционально) Google Search Console для данных о трафике

## Установка
